Hardware Projects

USB4GC - USB controller adapter for the Nintendo Gamecube

github.com/BadRAM/USB4GC

I created an adapter which can connect a usb game controller to a Nintendo Gamecube or Wii through it's own controller port.

In pursuit of this I reverse engineered the gamecube controller serial protocol, implemented it on a samd21 microcontroller, implemented a usb host on the microcontroller, and designed custom PCB and 3d printable plastic shell.

Who's afraid of PLA? - Battlebot

As an excercise in parametric CAD and integration, I designed and built a tanksteer robot chassis from 3d printed parts. The highly parameterized and 3d printable drivetrain made for fast and cheap iteration, and a custom power distribution PCB helped keep the wiring compact.

Mech Throttle - Custom input device

Inspired by 'joysticks on tracks' from cartoon robots, I built a custom gamepad with a single oversized analog control axis. pictured is the pure throttle configuration, but the handle can be easily swapped for a mounting plate which carries standard flight stick.
